# KONGSI 共享租赁平台

## 项目简介
KONGSI 是一个基于 Flutter 开发的共享租赁平台，支持物品浏览、详情查看、发布上传等功能，面向用户提供便捷的共享租赁服务体验。

## 个人职责
- 负责 Item Details Page 与 Upload Item Page 的页面设计与功能实现
- 集成 Firebase Storage，实现图片上传与云端存储
- 使用 Firebase Firestore 存储物品信息，完成前端表单与数据库字段映射
- 参与页面调试与交互优化，提升信息提交流程的完整性和可用性

## 技术栈
- Flutter
- Dart
- Firebase
- Firestore
- Firebase Storage

## 主要功能
- 物品详情展示
- 物品信息上传
- 图片上传与存储
- 数据持久化管理

## 说明
本项目为课程/毕业设计项目的一部分，主要用于展示移动端开发与 Firebase 集成能力。
